##### 辞書型（Python以外の言語では連想配列、ハッシュなどとも）
インデックスの概念がなく、キーを指定してバリューを取り出す
``` python
num_dict = {'りんご': 100, 'みかん': 120, 160: '梨'}
# 辞書[取り出したい要素のキー]でバリューを取り出す
print(num_dict['りんご']) # 100
```

##### タプル型

リスト型にはできない「辞書のキーに代入」ができる

##### セット型
複数の値を「集合」として扱う
- 一つのセットに重複した値を持たない
- 順序はなく、インデックスの概念もない
- 数学的な集合としての演算（和集合など）ができる
``` Python
n_set = {3, 'taro', 2.5}
print(n_set) # {'taro', 2.5, 3}
```
