## nonzero
`non-zero`: ゼロ以外の要素。真偽値の偽は0とみなされる為、Trueもnon-zeroとなる。
`np.nonzero()`
ゼロ以外の要素のインデックスを返す。
- `np.count_nonzero()`はゼロ以外の要素数をカウントする。
インデックスは行軸と列軸で分けられたタプルで返される。
- `np.transpose()`を使うことで、各インデックスを[行, 列]ペアの配列に転置できる。
```python
x = np.array([[3, 0, 0], [0, 4, 0], [5, 6, 0]])
x
'''
array([[3, 0, 0],
       [0, 4, 0],
       [5, 6, 0]])
'''
np.nonzero(x)　# (array([0, 1, 2, 2]), array([0, 1, 0, 1]))
x[np.nonzero(x)] # array([3, 4, 5, 6])
np.transpose(np.nonzero(x))
'''
array([[0, 0],
       [1, 1],
       [2, 0],
       [2, 1]])
'''
```
## transpose
`np.transpose(a, axis)`
軸の順序を変える（axisで順序を指定）
デフォルト（axis無指定）では順序が逆になる。
1次元配列では変化しない。

_1次元配列の場合、これは元の配列の視点を変更せずに返します。なぜなら転置ベクトルは単に同じベクトルだからです。1次元配列を2次元の列ベクトルに変換するには、追加の次元を追加する必要があります。例えば、np.atleast_2d(a).T や a[:, np.newaxis] がこれを実現します。2次元配列の場合、これは標準的な行列の転置です。n次元配列の場合、軸が指定されると、その順序が軸の置換方法を示します（例を参照）。軸が指定されない場合、transpose(a).shape == a.shape[::-1] となります。_
```python
# 1次元配列
a = np.array([1, 2, 3, 4])
a # array([1, 2, 3, 4])
np.transpose(a) # array([1, 2, 3, 4])

# 2次元配列
a = np.array([[1, 2], [3, 4]])
a
'''
array([[1, 2],
       [3, 4]])
'''
np.transpose(a)
'''
array([[1, 3],
       [2, 4]])
'''

# N次元配列
a = np.ones((1, 2, 3))
np.transpose(a, (1, 0, 2)).shape # (2, 1, 3)
```
