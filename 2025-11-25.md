## df.apply()の挙動
```python
def calc_bmi(row):
  weight = row['体重']
  height = row['身長']
  return weight / ( height / 100 ) ** 2
  
df["BMI"] = df[['体重', '身長']].apply(calc_bmi, axis=1)
df
```
calc_bmi関数の引数がrow1つのみで成り立つ理由。
1. apply()において、`axis=1`指定により関数へ'体重', '身長'の値を持つSeries型がindex順に渡され実行される。
2. calc_bmi関数では、引数によって渡されたSeries型(row)の'体重', '身長'データを使用して計算結果を返す。
3. 1~2の処理をindex数分繰り返し`df['BMI']`に代入する。
axis=1によって、1列を参照したSeries型が得られるというのがポイント。

## 区画分けとラベル付け
```python
df['身長カテゴリ'] = pd.cut(df['身長'],
						 bins=[0,160, 170, 250],
						 labels=['低', '中', '高'],
						 right=False)
```
binsリスト：境界値
labelsリスト：区画名（binsリストの要素数より-1）
right：境界値の右側を区画に含むか否か。
- True：0以上160以下。`[0:160]`
- False : 0 以上160未満。`[0:160)`
