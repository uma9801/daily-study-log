# Python
## ライブラリ: 特定の目的のためにパッケージ（モジュール）をまとめたもの
ライブラリを使うとに複雑な処理が簡単に実装できる。  
- パッケージ：ライブラリとモジュールの中間。
	- モジュール：複数のクラス・関数をまとめたもの。
		- 関数
		- クラス
			- メソッド
			- アトリビュート
				- インスタンス
### ライブラリの種類
- 標準ライブラリ：デフォルトでPythonに組み込まれているライブラリ。  
- 任意でインストールし、必要な時に読み込んで使用するライブラリ。  
	- `pip`コマンドでインストールする。  
インストールできるライブラリが豊富にあることがPythonの長所。
### ライブラリの使い方
`import ライブラリ名`で読み込む。  
`ライブラリ名.関数名`で使用する。  
`from ライブラリ名 import 関数名`で特定関数のみを読み込める。  
`import ライブラリ名 as 任意の略称名`で使用する略称名を設定できる。  
```python
import math # 標準ライブラリmathを読み込み

# mathライブラリのlog関数を使用
# 2を底としたlog8を求める
print(math.log(8, 2)) # 3.0
```
### 機械学習で使われるライブラリ
AI開発ではデータの加工や可視化が必要。  
そのために使われる主要なライブラリがNumpy, Pandas, Matplotlib  
Numpy：数値計算ライブラリ  
- 行列計算などを効率的に行える  
Pandas：データ解析を支援する機能を提供するライブラリ  
- データの前処理や管理を効率的に行える  
Matplotlib：グラフ描画ライブラリ  
- グラフを簡単に描画できる  
Scikit-learn（サイキット・ラーン）：機械学習ライブラリ  
- 機械学習に関わる処理を簡単に実装できる  
- SciPy(Scientific Python)という数値解析ソフトウェア・ライブラリのサードパーティ拡張(Toolkit)であることを示している名前
