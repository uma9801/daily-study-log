## インデックスの追加方法
- 単一のインデックスに新たなインデックスを追加してマルチインデックスにする  。
- `DataFrame.index = [追加するインデックスリスト, DataFrame.index]`  
	- DataFrame全体を扱う必要はなく、直接indexを指定して追加すれば良い。  
## 特定のデータの数を調べる
- `len(DataFrame[DataFlara[条件式]]`  
	- 例：`len(data[data["国語" >= 30]])`  
## 特定のデータについて調べる
- 国語の列で最も高い点数を持つ行のnameを調べる。  
	- `data[data["国語"] == data["国語"].max]["name"]`。  
		- `data["国語"] == data["国語"].max`で「国語でmax値を持つ」条件の国語列ブール値リスト（Series型）を作る。  
		- `data[data["国語"] == data["国語"].max]`で↑のリストでTrueを持つ行を抽出する。  
		- 最後に`["name"]`で↑の行でnameの列を抽出する。  
			- `DataFrame[列名]`のDataFrame部分が`data[data["国語"] == data["国語"].max]`ということ。    
	- 類似例：`data[data["数学"] >= 60][["name", "英語", "国語"]]`
		- DataFrameが`data[data["数学"] >= 60]`、[列のリスト]が`[["name", "英語", "国語"]]`ということ。  
## Series型、DataFrame型への関数の適用
- 各要素への関数の適用`map()`  
	- Series型、DataFrame型両方に使え、要素1つ1つに関数を適用する。  
	- `return num > 50`など、値個別に適用できる関数に向く。  
	- mapは数学の写像（mapping）から来ている「写像する」「対応させる」という意味。
- 列単位、行単位への関数の適用`allay()`  
	- DataFrame型に使われる。  
	- 列(行)毎に引数として関数を適用するので、下記のようなSeries型を引数とした関数などに向く。  
	- ```python
		def series_sum(s):
		if isinstance(s, pd.Series):
			return s.sum()
		else:
			return f"Series型の要素ではありません: {s}"
		```
	- applyは「適用する」という意味。
